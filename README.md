# Practice Python Exercises

My solutions to the exercises from [Practice Python](https://www.practicepython.org/exercises/), a free resource that offers beginner-friendly Python practice problems.

## About

This repository contains a Jupyter Notebook (`practicepython.ipynb`) with solutions to **40 exercises** from the Practice Python website, covering core Python fundamentals such as:

- Conditionals and loops
- Strings and string manipulation
- Lists, dictionaries, and tuples
- User input and basic I/O
- Functions and basic logic problems

Each exercise includes:
- The original problem description (as markdown)
- My code solution
- Any "Extras" / bonus challenges, where attempted

## Source

All exercise prompts are credited to **Practice Python**: https://www.practicepython.org/exercises/

## Usage

Clone the repo and open the notebook in Jupyter:

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
jupyter notebook practicepython.ipynb
```
